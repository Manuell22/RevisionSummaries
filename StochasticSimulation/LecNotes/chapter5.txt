5.1.1 the expectation-maximisation algorithm
A standard method to learn parameters in the presence of hidden, latent, or unobserved variables is the Expectation-Maximisation (EM) algorithm. The EM algorithm is a well-established
algorithm, first proposed by Dempster et al. (1977) and widely studied in several forms. In this
text, we will introduce the main template of the algorithm and prove the ascend property to show
that the EM algorithm can maximise the likelihood (in the absence of local minima).
The EM algorithm aims at producing iterates (θt)t≥0 that converge to a local maximum of the
marginal likelihood. Given a parameter estimate θt−1, the EM algorithm is a two-step iterative
algorithm that alternates between two steps to obtain the next estimate:
1. Expectation step (E-step): Compute the expected value of the log likelihood with respect
to the posterior distribution of the latent variables: Q(θ, θt−1) = Epθt−1
(x|y)
[log pθ(x, y)].
2. Maximisation step (M-step): Maximise the expected log likelihood with respect to the
parameters: θt = arg maxθ Q(θ, θt−1).

5.2.1 sampling from the posterior p(θ, x|y)
A simple way to sample from p(θ|y) is to sample from the joint posterior p(θ, x|y) and then
marginalise out x. This can be done with any of the sampling methods we have seen so far. For
example, we can run a Metropolis-Hastings algorithm targeting p(θ, x|y) and then use the θsamples to approximate the posterior p(θ|y). Similarly, we can run a Langevin dynamics targeting
p(θ, x|y) and then use the samples of θ to approximate the posterior p(θ|y). We can also run
a Gibbs sampler by alternating between sampling θ given x and sampling x given θ. The main
point is that we need to sample from the joint posterior p(θ, x|y) and then use these samples to
approximate the posterior p(θ|y).

5.2.2 pseudo-marginal metropolis-hastings
Another way to sample from p(θ|y) is to use pseudo-marginal MCMC methods. The idea is to
construct an unbiased estimator of the marginal likelihood p(y|θ) and then use this estimator in
a Metropolis-Hastings algorithm to sample from p(θ|y). This method is particularly useful when
the marginal likelihood is intractable but we can obtain an unbiased estimator of it, e.g., with
importance sampling. The main advantage of this method is that it allows us to sample from
p(θ|y) without having to sample from the joint posterior p(θ, x|y), which can be computationally
tricky when x is high-dimensional. However, the main challenge of this method is that the
variance of the estimator can be very high, which can lead to slow convergence. Therefore, it is
important to design good estimators of the marginal likelihood that have low variance.
The basis of this method is the fact that the MH algorithm remains valid if one replaces
p¯?(·) in the usual acceptance ratio with its unbiased estimate pb?(·) (Andrieu and Roberts, 2009).
This version of the method is termed pseudo-marginal Metropolis-Hastings (PMMH). This is
particularly useful when p¯?(·) is intractable to evaluate, which will be our setting.
